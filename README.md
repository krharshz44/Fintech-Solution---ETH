## 🧠 Overview

**AI Money Mentor 2.0** is a full-stack AI-powered fintech web application that acts as a **personal financial advisor** for users across all age groups.

It analyzes income, expenses, savings, and financial goals to generate:

* 📊 Money Health Score
* 📈 Investment Plans (SIP-based)
* ⚠️ Risk Analysis
* 🤖 AI-driven financial advice

Built with a focus on **real-world usability, simplicity, and intelligent automation**.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-money-mentor.git
cd ai-money-mentor
```

---

### 2️⃣ Setup Backend

```bash
cd server
npm install
```

Create `.env` file:

```env
MONGO_URI=your_mongodb_uri
OPENAI_API_KEY=your_openai_key
JWT_SECRET=your_secret
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Setup Frontend

```bash
cd client
npm install
```

Create `.env` file:

```env
VITE_API_URL=http://localhost:5000/api
```

Run frontend:

```bash
npm run dev
```
